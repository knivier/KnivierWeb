Project Description

Activerse is a powerful and versatile game engine designed to empower developers and creators to bring their imaginative worlds to life. With its intuitive interface and robust features, Activerse streamlines the game development process, offering seamless integration of physics, graphics, and sound. Whether you're a seasoned developer or a novice, Activerse provides the tools and flexibility to create complex, interactive experiences. Its modular architecture allows for easy customization and scalability, making it suitable for a wide range of game genres. Dive into Activerse and start crafting unique gaming experiences that captivate and engage players.

v1.1.2: Optimizations, QOL Improvements & Bug Fixes (Latest)

Changelog:

  • Logging Changes:
    • New log session started is always included first
    • Log parts are number tracked
    • FPS is logged actively
    • TargetFPS included with activeFPS
    • MB/s instead of MB/sec for consistency
    • Fixed bug where FPS logging only occurs when debug menu is open (now independent)
  • Debug Menu:
    • MB/s instead of MB/sec for consistency
  • Engine Changes:
    • Optimized World class for better RAM usage
    • Optimized some of Actor class for better RAM usage
    • Fixed calculation issues for FPS using GameLoop.java
    • TargetFPS is now a locked FPS which the game cannot exceed
    • Fixed collisions in Actor
  • Other Changes:
    • Code formatting and optimized imports
    • Changed some variables for clarity
